# KittyCats

A Lean formalization of undergraduate category theory built from first principles (i.e no Mathlib).

Categories are just like cats lounging on objects connected by arrows, and this library formalizes all the way up to cartesian closed categories.

- Steve Awodey, _Category Theory_ (2nd edition, Oxford Logic Guides, 2010).
- Saunders Mac Lane, _Categories for the Working Mathematician_ (2nd edition, Springer GTM, 1998).
